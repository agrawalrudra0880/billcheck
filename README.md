# BillCheck

BillCheck is an app that analyzes hospital bills to detect overcharges and generate objection reports.

## 🚀 Vision
Help patients identify billing fraud and save money by challenging incorrect charges.

## 🛠 Tech Stack
- Backend: Python (FastAPI)
- Frontend: (to be added)

## 📦 Features (MVP)
- Upload hospital bill (PDF/image)
- Analyze bill using rules/AI
- Generate report of suspicious charges
- Provide objection message template
